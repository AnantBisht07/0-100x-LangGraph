Questions: 
Where does state come from?	LangGraph creates it from your invoke() input
How do agents get state?	LangGraph passes it as the function argument
How do agents update state?	Their return dict gets merged into state
Is it the same state object?	Yes — one state flows through all agents
Who manages the flow?	LangGraph, based on the edges you defined
The agents don't call each other directly. LangGraph is the orchestrator — it runs each agent in order and handles passing/merging the state.
